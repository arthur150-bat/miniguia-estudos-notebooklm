# Caderno Temático com NotebookLM

## 1. Contexto e Objetivos

1. Contexto e Objetivos

Este projeto foi desenvolvido como parte de um desafio da DIO com o objetivo de explorar o uso da Inteligência Artificial como ferramenta de aprendizagem ativa utilizando o NotebookLM.

O tema escolhido foi Engenharia de Prompts, com o objetivo de compreender como formular perguntas mais eficientes para extrair melhores respostas de modelos de IA.

Objetivos de estudo:

Entender o que é engenharia de prompts
Aprender a estruturar perguntas para IA
Melhorar a qualidade das respostas obtidas

## 4. Dificuldades Encontradas

Durante o uso da IA foi observado que:

Perguntas muito genéricas geravam respostas superficiais.
Foi necessário ajustar os prompts para obter respostas mais detalhadas.
Prompts mais específicos geraram respostas mais úteis.

Esse processo de tentativa e erro ajudou a melhorar a qualidade das interações com a IA.

## 5. Miniguia de Estudo

### Resumo do Tema

A engenharia de prompts é a prática de criar instruções eficazes para sistemas de inteligência artificial com o objetivo de obter respostas mais precisas e úteis.

Essa técnica é importante para maximizar o potencial de modelos de linguagem.

### Glossário

IA – Inteligência Artificial que simula processos de inteligência humana.

Prompt – instrução ou pergunta fornecida a um modelo de IA.

LLM – Large Language Model, modelos de linguagem treinados com grandes quantidades de texto.
